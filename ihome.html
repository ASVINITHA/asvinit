iHOME
Overview
During my internship, I was given an important responsibility to design a static prototype for an upcoming website called iHome. The main goal was to create a website that was easy to use and navigate for both administrators and users. I took inspiration from similar existing systems to guide the design process. My role involved creating a detailed plan that outlined the website's features, functions, and how administrators and users would interact with it. After finalizing the system flow, it would be submitted to the officers for review and presentation to higher-ranking officials. During the presentation, the flow would be thoroughly discussed, and any feedback or requested changes would be addressed. This meeting served as an opportunity to ensure that the system flow aligned with the desired requirements and expectations. If any modifications were requested during the meeting, they would be implemented accordingly to meet the officers' specifications and create a finalized and refined system flow.
